Images

Images on the web are expected to be responsive because of the many screen sizes we are accessing websites from. In the bootstrap framework that we use, responsive images start with a width of 100% and an auto height. This means the the image will scale with the column size that the image is nested inside of.

When preping images for web we need to figure out the largest common denominator the image needs to be to fill the column that the image sits inside of. For starters, Bootstrap's largest media query starts at 1200px for it's content containers. Background images will scale to the width and height of the browser.

Images for web should always be prepped at 72 DPI.
